如题。

### 高票答案1

可以使用 [Kivy](http://kivy.org/)，目前在 [Github](https://github.com/kivy/kivy) 上开源。

> Kivy 可以运行在 Linux , Windows , OS X，Android 和 IOS。你可以在不同的平台上使用同一套的 python 代码。
